title: Overview
---

AIMP is a serverless-native autonomous database for building accurate, secure, and scalable AI applications.
Use AIMP to store and search unstructured data at scale without managing any infrastructure.

AIMP gives you everything you need for retrieval:

* Store text, embeddings, and their metadata
* Vector search
* Full-text search
* Hybrid search
* Metadata filtering

In addition, AIMP provides a set of convenient features to help you manage your data:

* Configurable project-level rate limiting
* Continuous backups and point-in-time recovery (PITR)
* Zero-copy index clone in seconds regardless of index size
* Zero operations due to its serverless-native nature
